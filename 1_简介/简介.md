
# 概述

    Hadoop 是分布式的存储和分布式计算框架
    
# 特点
    
    Hadoop使用HDFS完成分布式存储
    Hadoop使用Mapreduce完成分布式计算


  
# 优点

HDFS

    大数据存储
        可存储的数据数量多 
        支持很大单个文件。     
        
    数据高可靠性
        数据可以设置多个副本
        
      
# 缺点

HDFS
      
    延迟  
    
        分钟级别
          
    不适合小文件存储
     
        占用NameNode大量内存 寻道时间超过读取时间
          
    只能追加不能修改 
         
         一个文件只能有一个写者 仅支持append
