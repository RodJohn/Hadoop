
# 要求

    通过用户的好友列表，给用户推荐他最可能认识的人员

# 数据

    user    friend  friend  friend
    A       B       C       D
    B       C       F
    C       A       B
    D       A       M
    F       M
    M       F
        
# 思路

    map     以用户关系(正序)为key  
            朋友value=0
            朋友的朋友value=1
    reduce  有value为0说明已经是朋友
            没有value为0，递增推荐值
